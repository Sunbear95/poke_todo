# Poke Todo Agent Harness

This directory contains the simplified universal agent harness applied to the Poke Todo project.

Poke Todo is a low-friction todo/habit app concept where completed daily actions become Pokemon-style encounters during a daily wrap-up. The MVP focuses on:

- daily task/habit checklist
- 2-3 likely reward silhouettes before wrap-up
- one completed item creates one encounter
- user-triggered capture attempts
- repeat encounters increasing capture probability
- collection/Pokedex-style captured creature view

## Important Scope Boundary

MVP excludes evidence upload, AI proof analysis, Pokeball consumption, currency/shop/inventory, battles, social features, and complex long-term project hierarchy.

Pokemon-style assets/names are acceptable for communicating a private demo concept, but public release should switch to original creatures.
